# SITESELECT - 餐饮智能选址分析工具

## 项目概述
SmartLocation 是一个专为餐饮创业者设计的智能选址分析工具，支持牛肉饼、卷饼、甜点、烘焙等不同餐饮类型的选址需求。

## 核心功能
- 📍 地理位置热力分析
- 👥 目标客户画像分析  
- 🏪 竞争对手分布分析
- 💰 成本效益预测模型
- 📊 行业特定指标定制

## 技术栈
- 前端: React + Mapbox
- 后端: Node.js + Express
- 数据库: PostgreSQL + PostGIS
- 部署: GitHub Pages + Serverless

## 快速开始
```bash
git clone https://github.com/alansun123/SITESELECT.git
cd SITESELECT
npm install
npm start
```
