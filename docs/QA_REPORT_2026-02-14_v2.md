## 执行命令（摘要）
```bash
python3 -m py_compile app/gui_app.py src/siteselect/cli.py
python3 src/siteselect/cli.py analyze --input examples/candidates.example.csv --weights examples/weights.example.json --out output/report.html
streamlit run app/gui_app.py --server.headless true --server.port 18502
```

## 关键验证结论
1. **Missing field**
   - 旧问题：`KeyError traceback`
   - 现状：可继续执行并给出数据校验提示，无崩溃

2. **Extreme weights**
   - 旧问题：任意极端值直接参与计算
   - 现状：权重会被限制到 [0,1] 并归一化，CLI输出提示信息

3. **GUI runtime**
   - 旧 blocker：无 streamlit
   - 现状：`.venv` 安装后已成功启动 GUI
