# RELEASE NOTES v0.1.0

## 发布定位
GUI-first MVP（CLI 辅助），面向餐饮选址快速评估。

## 本版亮点
- GUI 四步流程：项目定义 / 数据导入 / 权重配置 / 结果导出
- 数据校验增强：空值、非数字、缺列场景可处理并提示
- 评分稳定性提升：同分场景下结果可复现
- 报告能力：支持 HTML 报告生成与下载
- 跨平台保障：新增 Windows + Ubuntu CI 检查工作流

## 关键改进
- 权重治理：CLI 对权重进行范围钳制（[0,1]）并自动归一化
- 数据质量可见：报告中展示“数据校验提示”列
- GUI 导出体验：时间戳文件名，避免覆盖

## 兼容性
- 已在 macOS 本地验证 CLI + GUI 启动
- 已配置 GitHub Actions Windows runner 自动校验（推送后执行）

## 已知限制
- 当前仍为本地文件输入（CSV/JSON），高德在线增强后续迭代
- 报告当前为 HTML，PDF/Excel 尚未纳入 v0.1.0
